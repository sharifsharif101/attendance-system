# تعليمات التثبيت في موقع العميل (On-Site Setup Instructions)

هذه الخطوات لك أنت (المبرمج) عندما تكون أمام جهاز العميل.

## 1. التثبيت الأولي
1.  ثبت برنامج **XAMPP** بشكل طبيعي في المسار الافتراضي `C:\xampp`.
2.  انسخ مجلد المشروع `attendance-system` إلى أي مكان (مثلاً سطح المكتب أو `C:\AttendanceApp`).
3.  قم بتثبيت الـ Dependencies (إذا لم يكن منقولاً جاهزاً):
    ```bash
    composer install
    npm install && npm run build
    ```
4.  أنشئ قاعدة البيانات في phpMyAdmin باسم `attendance_db`.
5.  شغل: `php artisan migrate --seed` لمرة واحدة.

## 2. تجهيز زر التشغيل السحري
1.  افتح ملف `OneClickSystem.bat` الموجود داخل مجلد المشروع.
2.  تأكد من أن مسار MySQL صحيح (السطر 9):
    ```bat
    set MYSQL_PATH=C:\xampp\mysql\bin\mysqld.exe
    ```
    (إذا كان XAMPP في مكان آخر، عدل هذا المسار فقط).

3.  **إنشاء الاختصار**:
    *   انقر بزر الماوس الأيمن على `OneClickSystem.bat` واختر **Send to -> Desktop (create shortcut)**.
    *   اذهب لسطح المكتب، وقم بتغيير اسم الاختصار إلى "نظام الحضور والانصراف" أو اسم الشركة.
    *   (اختياري) غير أيقونة الاختصار (Properties -> Change Icon) واختر أيقونة معبرة.

## 3. تسليم العميل
الآن، العميل يحتاج فقط للضغط مرتين على الأيقونة في سطح المكتب.
- سيقوم السكربت بتشغيل قاعدة البيانات (في الخلفية).
- سيقوم بتشغيل السيرفر.
- سيفتح المتصفح تلقائياً.
